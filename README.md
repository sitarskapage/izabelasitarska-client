# Lab Guy Client

## Installation

1. `npm i`
2. Create/add in `.gitignore` file in root directory
   ```js
    # labguy
    .env
   ```
3. Create/add variables in `.env` file in root directory
   ```bash
   VITE_SERVER_API_URL="http://localhost:3000/api"
   VITE_CLD_CLOUD_NAME="dzsehmvrr"
   ```
4. You are good to go!
   ```js
   npm run dev
   npm run deploy
   ```
